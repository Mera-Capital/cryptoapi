# CryptoAPI  
  
A multi-user library for processing a large number of transactions on different exchanges and with different tools  
  
### Supported exchanges  

#### **Binance**  
- Futures **USD-M**  
- Futures **COIN-M**  
- Spot  

#### **Deribit**  
- Futures  
- Spot  
  
### Supported methods

#### **Private**  
- get_position  
- get_balance  
- get_operations  
- close_all_orders  
- close_position  
- check_creds  
- buy  
- sell 

#### **Public**  
- get_all_history_data
- get_price  
- get_instruments  
- get_usd_index_price
